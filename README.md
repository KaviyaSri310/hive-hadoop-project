# Hive Hadoop Sales Analysis Project

This project analyzes sales data using Apache Hive on Hadoop.

## Technologies Used
- Hadoop
- Hive
- Linux
- Python
- Git & GitHub

## Project Structure

input/
- sales_data.csv

hive_scripts/
- sales_queries.hql

outputs/
- city_sales/city_sales.csv
- product_sales/product_sales.csv
- top_customers/top_customers.csv

## What this project does

✔ Total sales by product  
✔ Total sales by city  
✔ Top customers by purchase
